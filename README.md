# CESURG — Plano Financeiro Nexus

Material de estudo do grupo **Financeiro** para a atividade prática do AIHub CESURG (Nexus Soluções Industriais).

## Conteúdo

- [`index.html`](index.html) — página completa para estudo e apresentação (GitHub Pages)
- Documentos em Markdown com plano, roteiro de apresentação e respostas para a banca

## Assistente IA

O site inclui um **chat com IA gratuita** (via [Puter.js](https://developer.puter.com/tutorials/add-ai-chatbot-to-your-website/)) no canto inferior direito. Na primeira pergunta pode pedir login gratuito no Puter — depois disso, o grupo pode perguntar sobre margem, frete, apresentação, banca etc.
